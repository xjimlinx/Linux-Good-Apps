# Linux-Good-Apps
本人备份的Linux好用的软件列表

## 一、音乐
- **lx-music（洛雪音乐）**

> Arch linux 下安装：

```bash
yay -S lx-music-desktop-bin 
```

## 二、文献（文档查看）
- **Zotero**

> Arch linux 下安装：

```bash
sudo pacman -S zotero-bin
```
插件：
[划句翻译插件-zotero-pdf-translate](https://github.com/windingwind/zotero-pdf-translate)

## 三、游戏

- **Steam**

> Arch linux 下安装：

```bash
sudo pacman -S steam
```
